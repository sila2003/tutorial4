 Tutorial 4 – Java OOP

This project is a simple Java console application that models a project team.

It demonstrates key Object-Oriented Programming concepts such as:

* Inheritance (Employee → Developer, Tester)
* Polymorphism
* Method overriding (@Override)
* Safe casting using instanceof
* final methods
* toString() and equals()

The program creates different employees, stores them in one list, and calculates the total team cost.

 Classes

* Employee (parent)
* Developer
* Tester
* ProjectTeam
* Main

 Run

Run the Main class to see the example scenario.
